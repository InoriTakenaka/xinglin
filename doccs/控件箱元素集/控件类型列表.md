# 控件类型列表

## 工具箱控件元素

工具箱中包含以下控件元素，可用于模板设计：

| 控件名称 | 控件类型 | 描述 |
|---------|---------|------|
| 文本框 | TextBox | 可编辑的输入框，用于显示和编辑文本内容 |
| 静态文本 | FixedText | 固定的标签文本，用于显示不需要编辑的文本 |
| 下拉框 | ComboBox | 选项选择器，用于从预定义选项中选择 |
| 日期选择 | DatePicker | 日期选择器，用于选择日期 |
| 复选框 | CheckBox | 勾选框，用于表示布尔值状态 |
| 直线 | Line | 装饰线条，用于分隔或装饰界面 |
| 矩形 | Rectangle | 矩形框，用于边框或背景 |
| 圆形 | Circle | 圆形/椭圆，用于装饰或特殊标记 |
| 图片 | Image | 图片展示，用于显示图片资源 |
| 表格 | Table | 数据表格，用于显示结构化数据 |
| 图表 | Chart | 统计图表，用于可视化数据 |

## 控件属性

### 通用属性

所有控件都具有以下通用属性：

- **ElementId**: 元素唯一标识符
- **Type**: 控件类型
- **DisplayName**: 显示名称
- **X**: X坐标
- **Y**: Y坐标
- **Width**: 宽度
- **Height**: 高度
- **IsSelected**: 是否选中

### 文本类控件属性（TextBox、FixedText）

- **Value**: 文本值
- **FontSize**: 字体大小
- **FontWeight**: 字体粗细
- **HorizontalAlignment**: 水平对齐方式
- **BindingPath**: 数据绑定路径

### 表格控件属性（Table）

- **BindingPath**: 数据绑定路径
- **Columns**: 表格列定义
  - **Name**: 列名
  - **Width**: 列宽
  - **BindingPath**: 列数据绑定路径

### 线条控件属性（Line）

- **LineColor**: 线条颜色
- **LineThickness**: 线条粗细

### 图片控件属性（Image）

- **ImagePath**: 图片路径
- **Stretch**: 拉伸方式

## 控件创建流程

1. 在工具箱中选择所需控件类型
2. 在模板编辑区域点击或拖拽放置控件
3. 根据需要调整控件属性（位置、大小、样式等）
4. 设置数据绑定路径（如果需要）
5. 保存模板

## 控件使用示例

### 文本框控件

用于显示患者信息、检验结果等动态数据：

```json
{
  "ElementId": "patient-name",
  "Type": 1,
  "DisplayName": "姓名",
  "X": 5,
  "Y": 95,
  "Width": 50,
  "Height": 20,
  "BindingPath": "Patient.Name",
  "Value": "姓名：{Value}",
  "FontSize": 12.0
}
```

### 表格控件

用于显示检验项目列表：

```json
{
  "ElementId": "test-table",
  "Type": 4,
  "DisplayName": "检验项表格",
  "X": 0,
  "Y": 80,
  "Width": 128,
  "Height": 280,
  "BindingPath": "Results",
  "Columns": [
    { "Name": "项目", "Width": 50, "BindingPath": "ItemName" },
    { "Name": "结果", "Width": 30, "BindingPath": "Value" },
    { "Name": "参考值", "Width": 48, "BindingPath": "ReferenceRange" }
  ]
}
```

### 直线控件

用于分隔不同区域：

```json
{
  "ElementId": "line-top",
  "Type": 2,
  "X": 0,
  "Y": 85,
  "Width": 180,
  "Height": 2
}
```
