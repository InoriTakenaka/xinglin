# 开发步骤

## 1. 环境搭建

### 1.1 安装Visual Studio 2022
- **版本要求**：Visual Studio 2022 17.8或更高版本
- **工作负载**：.NET桌面开发
- **安装步骤**：
  1. 下载Visual Studio 2022安装程序
  2. 运行安装程序，选择.NET桌面开发工作负载
  3. 等待安装完成

### 1.2 安装.NET 8 SDK
- **版本要求**：.NET 8.0 SDK或更高版本
- **安装步骤**：
  1. 访问Microsoft官网下载.NET 8.0 SDK
  2. 运行安装程序
  3. 等待安装完成
  4. 验证安装：打开命令提示符，运行 `dotnet --version`

### 1.3 创建项目
- **项目类型**：WPF应用程序
- **框架**：.NET 8.0
- **创建步骤**：
  1. 打开Visual Studio 2022
  2. 点击"创建新项目"
  3. 选择"WPF应用"
  4. 点击"下一步"
  5. 输入项目名称和位置
  6. 选择.NET 8.0框架
  7. 点击"创建"

### 1.4 添加依赖项
- **必要的NuGet包**：
  - CommunityToolkit.Mvvm：用于MVVM模式实现
  - Microsoft.Extensions.DependencyInjection：用于依赖注入
  - System.Text.Json：用于JSON序列化和反序列化
- **安装步骤**：
  1. 右键点击项目，选择"管理NuGet包"
  2. 点击"浏览"选项卡
  3. 搜索并安装上述NuGet包

## 2. 项目结构搭建

### 2.1 创建目录结构
- **主要目录**：
  - `Models/`：数据模型
  - `ViewModels/`：视图模型
  - `Views/`：视图
  - `Services/`：服务
  - `Services/Data/`：数据服务
  - `Services/Pdf/`：PDF服务
  - `Assets/`：资源文件
  - `Assets/baseTemplate/`：基础模板
  - `Utils/`：工具类

### 2.2 配置文件
- **appsettings.json**：应用程序配置
- **创建步骤**：
  1. 右键点击项目，选择"添加" > "新建项"
  2. 选择"JSON文件"
  3. 命名为"appsettings.json"
  4. 点击"添加"
