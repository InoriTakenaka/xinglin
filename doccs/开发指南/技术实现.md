## 2. 核心数据结构

### 2.1 TemplateData
- **描述**：模板数据模型，包含模板的所有信息
- **核心属性**：
  - `TemplateId`：模板唯一标识符
  - `Name`：模板名称
  - `Description`：模板描述
  - `Layout`：布局信息
  - `Config`：配置信息
  - `CreatedDate`：创建时间
  - `ModifiedDate`：修改时间
  - `ParentId`：父模板ID
  - `ChildTemplateIds`：子模板ID列表
- **实现**：
  ```csharp
  public partial class TemplateData : ObservableObject
  {
      [ObservableProperty]
      private string _templateId;

      [ObservableProperty]
      private string _name;

      [ObservableProperty]
      private string _description;

      [ObservableProperty]
      private LayoutMetadata _layout;

      [ObservableProperty]
      private TemplateConfig _config;

      [ObservableProperty]
      private DateTime _createdDate;

      [ObservableProperty]
      private DateTime _modifiedDate;

      [ObservableProperty]
      private EditorState _editorState;

      [ObservableProperty]
      private string _version;

      [ObservableProperty]
      private string _parentId;

      [ObservableProperty]
      private List<string> _childTemplateIds;

      public TemplateData()
      {
          TemplateId = Guid.NewGuid().ToString();
          CreatedDate = DateTime.Now;
          ModifiedDate = DateTime.Now;
          EditorState = EditorState.DesignMode;
          Version = "1.0";
          ChildTemplateIds = new List<string>();
      }
  }
  ```

### 2.2 LayoutMetadata
- **描述**：布局元数据，包含模板的布局信息
- **核心属性**：
  - `PaperType`：纸张类型
  - `IsLandscape`：是否横向
  - `PaperWidth`：纸张宽度
  - `PaperHeight`：纸张高度
  - `MarginLeft`：左边距
  - `MarginRight`：右边距
  - `MarginTop`：上边距
  - `MarginBottom`：下边距
  - `FixedElements`：固定元素列表
  - `EditableElements`：可编辑元素列表

### 2.3 ControlElement
- **描述**：控件元素，是所有控件的基类
- **核心属性**：
  - `ElementId`：元素唯一标识符
  - `Type`：控件类型
  - `DisplayName`：显示名称
  - `X`：X坐标
  - `Y`：Y坐标
  - `Width`：宽度
  - `Height`：高度
  - `IsSelected`：是否选中

### 2.4 TableElement
- **描述**：表格控件元素，继承自ControlElement
- **核心属性**：
  - `BindingPath`：数据绑定路径
  - `Columns`：表格列定义
    - `Name`：列名
    - `Width`：列宽
    - `BindingPath`：列数据绑定路径

### 2.5 ReportInstance
- **描述**：报告实例，包含报告的所有信息
- **核心属性**：
  - `ReportId`：报告唯一标识符
  - `TemplateId`：模板ID
  - `TemplateVersion`：模板版本
  - `Data`：报告数据
  - `CreatedDate`：创建时间
  - `ModifiedDate`：修改时间

## 3. 关键服务实现

### 3.1 ITemplateService
- **描述**：模板服务接口，定义模板管理的所有方法
- **核心方法**：
  - `GetAllTemplatesAsync()`：获取所有模板
  - `GetTemplateByIdAsync(string templateId)`：根据ID获取模板
  - `SaveTemplateAsync(TemplateData template)`：保存模板
  - `DeleteTemplateAsync(string templateId)`：删除模板
  - `LoadDefaultTemplateAsync()`：加载默认模板
  - `ImportTemplateAsync(string sourceFilePath)`：导入模板
  - `LoadTemplateFromFileAsync(string filePath)`：从文件加载模板
  - `CreateNewTemplate()`：创建新模板
  - `SaveReportInstanceAsync(ReportInstance reportInstance)`：保存报告实例
  - `LoadReportInstanceAsync(string reportId)`：加载报告实例
  - `GetReportInstancesAsync()`：获取所有报告实例
  - `DeleteReportInstanceAsync(string reportId)`：删除报告实例
  - `GetTemplates()`：获取模板树结构
  - `GetTemplatesTreeAsync()`：获取模板数据树结构
  - `OpenTemplate(string templatePath)`：打开模板
  - `GetCurrentTemplateData()`：获取当前模板数据
  - `SaveTemplateData(TemplateData templateData)`：保存当前模板数据

### 3.2 TemplateService
- **描述**：模板服务实现，负责模板的实际管理
- **核心实现**：
  - **构造函数**：初始化服务，设置目录路径，确保目录存在
  - **模板加载**：从文件系统加载模板，支持缓存
  - **模板保存**：将模板保存到文件系统，更新缓存
  - **模板导入导出**：支持模板的导入和导出
  - **报告实例管理**：管理报告实例的增删改查
  - **模板树结构**：构建和获取模板的树结构

### 3.3 IDataService
- **描述**：数据服务接口，定义数据管理的所有方法
- **核心方法**：
  - `SaveDataAsync(object data, string key)`：保存数据
  - `LoadDataAsync(string key)`：加载数据
  - `ValidateData(object data)`：验证数据
  - `GenerateReport(TemplateData template, object data)`：生成报告
  - `ImportData(string filePath)`：导入数据
  - `ExportData(object data, string filePath)`：导出数据

### 3.4 DataService
- **描述**：数据服务实现，负责数据的实际管理
- **核心实现**：
  - **数据保存**：将数据保存到文件系统
  - **数据加载**：从文件系统加载数据
  - **数据验证**：验证数据的有效性
  - **报告生成**：根据模板和数据生成报告
  - **数据导入导出**：支持数据的导入和导出
